# Version Roadmap

## V1: Course Map And Module Summary

- Complete the course learning site foundation.
- Display 17 modules, sections, and topics.
- Support search and local learning progress.
- Complete responsive layout.

## V1.1: Structured Full Text

- Structure docx正文 into pages.
- Support paragraphs, explanations, and notes.
- Start handling code snippets.

## V1.2: Code Blocks And Example Runtime Area

- Display important code examples.
- Support code copy.
- Support example run explanations or an embedded run area.
- Do not add real online Python execution unless explicitly requested and designed.

## V1.3: Exercises And Answer Explanations

- Add exercises for each module.
- Support multiple choice, fill-in-the-blank, and code reading questions.
- Support answer explanations and incorrect-question marking.

## V2: Content Management Or API Data

- Migrate static course data to API data.
- Support backend maintenance for modules, sections, topics, and正文.
- Support multiple courses.
- Support user accounts and cloud learning progress.

## Non-Goals Unless The Version Requires Them

- No backend in V1.
- No login or registration in V1.
- No cloud progress sync in V1.
- No full docx migration in V1.
- No docx image/table processing in V1.
- No online Python code execution in V1.
- No backend course management in V1.
